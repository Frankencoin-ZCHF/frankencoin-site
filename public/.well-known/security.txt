[ZCHF is an ERC20 token pegged to the Swiss franc. Dig into the coin's smart contract security and get rewarded. +teaser+]()

Frankencoin (ZCHF) is a collateralized, oracle-free stablecoin that tracks the value of the Swiss franc. The coins strengths are its decentralization and its versatility. It is an ERC20 token and available on 5 chains: Ethereum, Polygon, Arbitrum, Optimism, and Base. The Frankencoin protocol shall be the best it can be, and the security of Frankencoin is the absolute highest priority. Thus, it was audited by leading security firms and Frankencoin calls on the community to help find bugs or vulnerabilities.

# Excluded Items

The bug bounty **exclusion list** comprises targets that are excluded from the program, and no rewards will be granted for reported vulnerabilities on these targets.

- Bugs in Frankcoin contract code which is not deployed in production is out of scope. Repository for reference https://github.com/Frankencoin-ZCHF/FrankenCoin
- Code forked from external repositories and forked files left untouched
- Code intended for demo, testing purpose (non-production code)
- Any 3rd Party-Services (M365, O365, Atlassian, Azure, Github etc.)
